HL²DM Pro can run maps from other mods natively

To do that, you only need to specify "path" to the mod in order maps to be seen by Adrenaline Gamer 2. Of course, the external mods must be installed in both the server and the client (player) sides.
Also, as Valve is now starting distributing mods through SteamWorks, you have to specify ExtraAppID of the mods if this one is in a gcf file so that its SteamContent will be loaded. To do that, just add \\ExtraAppId after mod name
Adrenaline Gamer 2 is compliant with all mods running HL2 Engine Episode 1 or Orange Box Engine.
