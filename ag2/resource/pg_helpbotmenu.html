RCBot2

RCBot2 is a bot system having its community in here : http://rcbot.bots-united.com
This BOT system needs a waypoint file per map. This waypoint file can be done using specific command of RCBot2. A tutorial is present in our website http://www.hl2dmpro.com


Control Panel

Control Panel has been done to easily interface with the plug-in without having to type console commands.

- Activate Bots: check this box to activate bots
- Tag: prefix to name of each bot
- Skill: skill level of bots (1 = very good, 10 = very bad)
- Min: minimum number of bots
- Max: maximum number of bots
- No bot if: bots will disappear when this number of REAL players is reached
- No bot if no waypoint: check this box to prevent bot to spawn in case a map has no waypoints file
