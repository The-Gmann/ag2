What is a Gameplay ?

A Gameplay is a set of parameters which does influence the game in terms of :
- weapons : damage, missile/bolt speed, clip size, allowed weapons, ...
- movement : speed, bunny hop or not, wall jump or not, ...
- specific features : hitsound, bright/regular models, longjump module, ...


The Gameplay SDK is composed of 3 panels

- Gameplay : speeds, specific features enabler
- Weapons 1 : damages
- Weapons 2 : push effect (Quakelike) and clip size



Gameplay Panel

When this panel is opened, the current gameplay is selected so that you can see current settings.

4 sections

- Movement
- Miscellaneous
- Speed
- Buttons


Creating a new Gameplay

You must be an administrator to be able to create a new gameplay. To check that you are admin, go to the "Admin Panel" (F11 by default) and check that "You are administrator" is written in green. Otherwise, fill in the server RCON password and click "submit". Then go back to the gameplay panel.

- Click on the zone next to "Gameplay". The zone should turn red.
- Enter the name of your choice : i.e. My gameplay
- When creating your own gameplay, it is easier to use an existing reference. The buttons "Set Original HL2MP" and "Set Pro" permits you to load the full set of parameters for either Regular HL2DM or Promod.
- Then you can alter the values in this panel to match your needs.

What these parameters do affect in term of gameplay is defined below :

Movement section
- Walk speed: velocity when you walk
- Normal speed: velocity when you run
- Sprint speed: velocity when you sprint
- Bunny hopping: check this box to allow bunny hopping ala "pro" (with on-air acceleration)
- Long jump: check this box to allow Jump Pack
- Wall jump: check this box to allow Wall jumps

Miscellaneous section
- Drop weapons: check this box to allow players to drop weapons
- 357 zoom: check this box to allow zoom for magnum
- RPG toggle guiding: check this box to allow rocket guiding to be disabled or not by the player in game
- Brightskins: check this box to activate brightskins. It is similar to accept or not 6 teams. You can only have 2 teams without brightskins.
- Combine ball self kill: check this box to make combine ball able to kill the shooter

Speed section
- Missile speed: speed of the missile
- Non-zoomed Air Bolt speed: speed of bolt when in air and not zoomed. Take care that bolt has a weight.
- Zoomed Air Bolt speed: speed of bolt when in air and zoomed. Take care that bolt has a weight.
- Non-Zoom Water Bolt speed: speed of bolt when in water and not zoomed. Take car that bolt has a weight.
- Zoomed Water Bolt speed: speed of bolt when in water and zoomed. Take car that bolt has a weight.

By clicking "Weapons" button you reach the section where you can alter the settings related to each weapon. The weapons section is pretty complex so it has been divided in 2 panels. Click on Help in each section to get explanation for each parameter.

- Click "Submit" to send your new gameplay defintion to the server. A message box must confirm the success of the transfer.
- Click on "Close" to close the "Gameplay" panel.

At this point in time you have created a gameplay but this gameplay is not active. To activate it, you have to ensure that it is votable and use it in your server. Proceed as follow :

- Open the Admin panel
- In the box "Settings" ensure that "Gameplay" is set to "Votable"
- Open the "Command Menu"
- In the "Settings" box, vote for "My gameplay" gameplay

Enjoy !!!


Exporting/Importing a Gameplay

Let's say that you like a gameplay available in a server you are not the admin and that you want to implement this specific gameplay in your server then here is how to proceed.

Exporting from server:

- Join the server with the gameplay you want to get
- Go to the "Admin Panel" (F11)
- Click on "Gameplay"
- Click on "Export from server" --> a message will inform you that the gameplay has been exported successfully and that it is locate in your cfg/gameplay folder

Send to your server:

- Join your server
- Go to "Admin Panel" (F11)
- Login as Administrator
- Click on "Gameplay"
- Click on "Send to server"
- Select the gameplay file you just exported in your cfg/gameplay folder --> a message will inform you that gameplay has been successfully exported
- Click "Close" button
- Open the Admin panel
- In the box "Settings" ensure that "Gameplay" is set to "Votable"
- Open the "Command Menu"
- In the "Settings" box, vote for the gameplay you just sent

Enjoy !!!
