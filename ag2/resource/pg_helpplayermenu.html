Colors in your Name

In AG2 you can use several colors in your Name. There are 2 ways:
- using ^x
- using $rgb


Way 1: ^x

- ^0: default color from Valve
- ^1: red
- ^2: green
- ^3: yellow
- ^4: blue
- ^5: cyan
- ^6: pink
- ^7: grey
- ^8: white
- ^9: orange
- ^a: black
- ^b: dark blue
- ^c: dark green
- ^d: dark red
- ^e: dark pink
- ^f: dark grey
- ^g: dark cyan

Way 2: $rgb

- r = red
- g = green
- b = blue
with a value going from 0 to F

ie: $F00 for full Red or $400 for Dark Red
